# 📊 Power BI Assignment 2 – DAX & Data Visualization

## 🚀 Project Overview

This project demonstrates the use of **Power BI** and **DAX (Data Analysis Expressions)** to analyze an E-Commerce Sales dataset. The report includes calculated columns, DAX measures, interactive dashboards, and analytical visualizations to generate meaningful business insights on sales performance, profitability, and regional trends.

---

## 🎯 Objectives

- Analyze e-commerce sales data using Power BI.
- Create calculated columns and measures using DAX.
- Track sales performance and profitability across product categories.
- Analyze sales distribution by city and state.
- Monitor monthly sales trends.
- Compare actual sales with sales targets.
- Build an interactive dashboard for business reporting and decision-making.
- Improve practical skills in data visualization and business intelligence.

---

## 🧮 DAX Calculated Columns

### ✅ Category Type
Combined **Category** and **Sub-Category** into a single column.

### ✅ Revenue per Order
```DAX
Revenue per Order = Amount * Quantity
```

### ✅ Sales Category

Categorized sales into:

- Above Average
- Below Average

---

## 📈 DAX Measures

### 📦 Order Count
Calculates the total number of orders.

### 📈 Average Profit in Delhi
Calculates the average profit for orders placed in Delhi.

### 📊 Year-to-Date (YTD) Sales
Calculates cumulative sales from the beginning of the year.
